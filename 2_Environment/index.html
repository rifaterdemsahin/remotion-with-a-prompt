☁️ GitHub Codespaces

Launch a full VS Code dev environment in your browser — no local install required. Perfect for quick edits, exploration, or working from any machine.

☁️ Open in Codespaces 🔀 Fork first (recommended)
1️⃣ Open Codespace

Click Code → Codespaces → Create codespace on main on the GitHub repo page, or use the button above.

2️⃣ Install deps

The terminal opens automatically. Run npm install if you want to use Remotion's CLI renderer.

3️⃣ Preview the app

Use Live Server or run npx serve . and click the forwarded port URL to open the studio.

4️⃣ Set your xAI key

Copy env-config.example.js → env-config.js and paste your key, or use the ⚙️ Settings panel in the app.

# In the Codespaces terminal:
cp env-config.example.js env-config.js
# Edit env-config.js and add your XAI_API_KEY

# Serve the static site locally (port forwarded automatically):
npx serve . -p 8080

# Optional: run Remotion studio
npm install
npx remotion studio 5_Symbols/src/Root.tsx
