# 🎨 Simulation — The "Vision"

> Stage 3 of 7: See it before you build it.

## Purpose

This folder contains UI mockups, animated previews, and design explorations for the Remotion Prompt Studio.

## Carousel

The `index.html` features a dynamic carousel showcasing:
1. Prompt Studio interface
2. Image loading flow
3. Live preview animation
4. Code output view
5. Test results dashboard

## Design Principles

- **Dark theme** — matches video editing tools (Premiere, DaVinci)
- **Purple accent** — aligned with Remotion's brand color (#7c3aed)
- **16:9 preview ratio** — standard video format
- **Minimal chrome** — focus on content, not decoration

## Mockups

See `mockups/` subfolder for wireframes.
Current simulations are rendered via Canvas API directly in the browser — no image files needed.
